## Template Rules

**Layout Pattern:**
- Labels ABOVE boxes (vertical stacking)
- Two columns side-by-side where space permits
- Full-width fields for long text (addresses, text areas)
- Empty boxes (no placeholder data)

**Dropdown Fields:**
- Show dropdown indicator: `│ ▼ Please Select │`
- List options directly below box
- Options naturally push fields below down

**Box Characters:**
```
┌─────────┐
│         │
└─────────┘
```

**This is the perfect clean format - use as reference for all future forms!**
