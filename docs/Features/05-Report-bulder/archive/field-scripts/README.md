## Purpose of These Scripts

These Python scripts were used to generate field analysis reports in early December 2025. They performed:

- Extraction of field IDs from Word HTML templates
- Comparison with fieldRegistry.ts definitions
- Generation of alignment reports and mapping tables
- Analysis of table structures and field distributions
