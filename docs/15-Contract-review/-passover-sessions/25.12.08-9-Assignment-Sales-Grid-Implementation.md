## Data Flow Pattern Established

```
Test Data (northBattlefordRealData)
    → fieldToSectionMap (maps field IDs to section IDs)
    → loadNorthBattlefordRealData() (loads into store)
    → Store (reportBuilderStore.ts with inputType classification)
    → Renderer (getFieldValue() retrieves values)
    → Preview HTML
```

**inputType Classification:**
- `user-input` (yellow) - Appraiser enters manually
- `dropdown` (blue) - Selected from options
- `auto-filled` (green) - From API/calculations
