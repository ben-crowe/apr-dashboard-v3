```
       │

ORCHESTRATOR READS ACTUAL OUTPUT
(Fast Filesystem MCP, line extraction, searches)
       │
       ├─ Verify checklist items with evidence
       ├─ Confirm changes present (line numbers)
       ├─ Check for regressions
       │
       ▼
REPORT TO USER:
✅ "Verified complete - all items confirmed with evidence"
OR
⚠️  "Issues found - items X, Y, Z missing - fixing now"
```

**Why this matters:**
- AI agents can hallucinate completion
- Tools can fail silently
- "Done" ≠ "Done correctly"
- User shouldn't be the QA for orchestrator's sub-agents
