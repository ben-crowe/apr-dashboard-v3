# Change User Type to External

**Issue:** App is set to "Internal" but needs to be "External"

**Fix:** Change on Audience page

---

## Step 1: Go to Audience Page

**In the left menu, click:**
- **"Audience"**

---

## Step 2: Change User Type

**On the Audience page:**

1. **Look for:** "User type" section
2. **Change from:** "Internal" 
3. **Change to:** "External"
4. **Click:** "Save" or "Update"

---

## Step 3: After Changing to External

**You'll need to:**

1. **Add test users:**
   - Go to "Test users" section
   - Add: `bc@crowestudio.com`

2. **Add scopes:**
   - Go to "Data Access" section
   - Add: `gmail.readonly` scope

3. **Make sure publishing status is "Testing"** (not "In production")

---

**Action:** Click **"Audience"** in the left menu, change to "External", and save!
