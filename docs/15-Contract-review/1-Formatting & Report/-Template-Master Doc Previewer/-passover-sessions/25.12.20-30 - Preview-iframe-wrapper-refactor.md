## 🎯 TECHNICAL IMPLEMENTATION DETAILS

### Final Architecture (Iframe Wrapper Pattern)

**PreviewPanel.tsx - Complete Refactor:**
```typescript
import { useEffect, useRef } from 'react';
import { useReportBuilderStore } from '../../store/reportBuilderStore';

export default function PreviewPanel() {
  const previewHtml = useReportBuilderStore((state) => state.previewHtml);
  const iframeRef = useRef<HTMLIFrameElement>(null);

  // Inject the report pages into the standalone viewer when HTML changes
  useEffect(() => {
    if (!previewHtml || !iframeRef.current) return;

    const iframe = iframeRef.current;

    const handleLoad = () => {
      const iframeDoc = iframe.contentDocument || iframe.contentWindow?.document;
      if (!iframeDoc) return;

      // Find the pages wrapper in the standalone viewer
      const pagesWrapper = iframeDoc.getElementById('pages-wrapper');
      if (!pagesWrapper) {
        console.error('pages-wrapper not found in preview-wrapper.html');
        return;
      }

      // Extract just the page-sheet divs from the template HTML
      const tempDiv = document.createElement('div');
      tempDiv.innerHTML = previewHtml;
      const pageSheets = tempDiv.querySelectorAll('.page-sheet');

      // Clear wrapper and inject pages
      pagesWrapper.innerHTML = '';
      pageSheets.forEach(page => {
        pagesWrapper.appendChild(page.cloneNode(true));
      });

      // Trigger page count update if the wrapper has that function
      const iframeWindow = iframe.contentWindow as any;
      if (iframeWindow && typeof iframeWindow.updatePageCount === 'function') {
        iframeWindow.updatePageCount();
      }
    };

    if (iframe.contentDocument?.readyState === 'complete') {
      handleLoad();
    } else {
      iframe.addEventListener('load', handleLoad);
      return () => iframe.removeEventListener('load', handleLoad);
    }
  }, [previewHtml]);

  return (
    <div className="h-full w-full">
      {previewHtml ? (
        <iframe
          ref={iframeRef}
          src="/preview-wrapper.html"
          style={{
            width: '100%',
            height: '100%',
            border: 'none',
            display: 'block'
          }}
          title="Report Preview"
        />
      ) : (
        <div className="h-full flex items-center justify-center">
          <p className="text-muted-foreground">Loading preview...</p>
        </div>
      )}
    </div>
  );
}
```

## 💡 KEY LEARNINGS

### Don't Fight Working Code
- **Learning:** User had a perfectly functional standalone HTML viewer
- **Mistake:** Tried to rebuild all controls in React instead of using what worked
- **Solution:** Iframe wrapper pattern - let proven code do its job
- **Takeaway:** When something works, integrate it, don't reimplement it

### Scroll Container vs Iframe Content
- **Learning:** In an iframe setup, the container scrolls, not the iframe content
- **Mistake:** Initially listened to iframe.contentWindow for scroll events
- **Solution:** Moved scroll listener to containerRef
- **Takeaway:** Understand which element actually scrolls before attaching listeners

### setState Callback Form for Intervals
- **Learning:** setInterval with state can create stale closures
- **Mistake:** Used `setCurrentPage(currentPage + 1)` which always referenced initial value
- **Solution:** Used `setCurrentPage(prevPage => prevPage + 1)` to get current value
- **Takeaway:** Always use callback form in intervals/timers to avoid stale state

### Debouncing Scroll Events
- **Learning:** Scroll events fire extremely rapidly
- **Mistake:** Updated state on every scroll event, caused glitches
- **Solution:** Added 150ms debounce to only update after scrolling stops
- **Takeaway:** Debounce high-frequency events before triggering state updates

### Template Cleanup is Critical
- **Learning:** Old code in templates can resurface unexpectedly
- **Mistake:** Removed React controls but forgot to clean template HTML
- **Solution:** Thoroughly searched and removed 140+ lines of old controls
- **Takeaway:** When refactoring, clean ALL related code, not just the component

### Architecture Over Band-Aids
- **Learning:** Multiple small fixes won't solve a fundamental architecture problem
- **Mistake:** Kept trying to fix React implementation instead of questioning approach
- **Solution:** Complete refactor to iframe wrapper after user insight
- **Takeaway:** When you keep fighting the same problem, step back and question the architecture
