# Manual Navigation to Data Access

**Don't use my URLs - navigate manually!**

---

## Step-by-Step Navigation

1. **You're currently on:** Audience page (you can see "Publishing status: Testing")

2. **In the LEFT MENU**, look for:
   - Overview
   - Branding
   - Audience (you're here)
   - Clients
   - **Data Access** ← Click this one!
   - Verification Center
   - Settings

3. **Click "Data Access"** in the left menu

---

## What You Should See

**On the Data Access page, you should see:**
- A list of scopes/permissions
- Or an option to add scopes
- Or it might be empty (which is fine - script will request it)

---

**Action:** Just click **"Data Access"** in the left menu (where you clicked "Audience" before)!
