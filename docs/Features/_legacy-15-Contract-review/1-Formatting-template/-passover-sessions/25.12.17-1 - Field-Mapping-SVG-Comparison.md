## Technical Reference

**Field-Mapped Span Format (CRITICAL):**
```html
<span class="field-mapped" data-sample="VALUE" title="{{Field_ID}}">{{Field_ID}}</span>
```

**Why this format:**
- `textContent` = `{{Field_ID}}` (shows field ID by default)
- `data-sample` = actual value (toggle feature reads this)
- `title` = `{{Field_ID}}` (tooltip reference)

**Toggle Mechanism:**
```javascript
// Toggle OFF: Shows {{Field_ID}}
// Toggle ON: field.textContent = field.getAttribute('data-sample');
```

**Common Field ID Patterns:**
- Market Rent: `{{Market_Rent_[1/2]Bed_Comp[1-5]_[Metric]}}`
- Revenue: `{{ContractVsMarket_[UnitType]_[Metric]}}`
- Comparables: `{{Comp[1-5]_[FieldName]}}`
- Comments: `{{OpEx_[Category]_Comment}}`

**CSS Utility Classes:**
- `.text-right` - Right-align text/numbers
- `.bold` - Bold text
- `.compact-table` - Compact table spacing
- `.label-col` - Label column styling
