## Quick Start

### 1. View Your History
```bash
# Navigate to SpecStory directory
cd .specstory

# List all sessions
./specstory-utils.sh list

# Show latest 5 sessions
./specstory-utils.sh latest 5

# Show statistics
./specstory-utils.sh stats
```

### 2. Search Your History
```bash
# Search for a term across all sessions
./specstory-utils.sh search "image configurator"

# Find sessions that mention a specific file
./specstory-utils.sh find-file "ImageConfiguratorDemo.tsx"

# Find sessions about a topic
./specstory-utils.sh find-topic "TDD Dashboard"
```

### 3. View a Specific Session
```bash
# Show full content of a session
./specstory-utils.sh show 2026-01-06_22-45Z-image-page-configurator-color-discrepancy.md

# Or use cat directly
cat history/2026-01-06_22-45Z-image-page-configurator-color-discrepancy.md
```

### 4. Use in Cursor AI

When chatting with Cursor, reference past sessions:

```
@.specstory/history/2026-01-06_22-45Z-image-page-configurator-color-discrepancy.md
```

This loads the full context from that session into your current conversation.

## Understanding Session Files

Each session file contains:

1. **Header**: Session metadata (timestamp, session ID, model)
2. **User Queries**: What you asked
3. **AI Responses**: What the AI suggested
4. **Tool Usage**: Detailed logs of:
   - Code searches performed
   - Files read
   - Code edits made
   - Terminal commands run
5. **Full Context**: Everything needed to understand the conversation

### Example Session Structure

```markdown
<!-- Generated by SpecStory, Markdown v2.1.0 -->
<!-- cursor Session abc123... (2026-01-06 22:45Z) -->

# Image page configurator color discrepancy (2026-01-06 22:45Z)

_**User (2026-01-06 00:58Z)**_
[Your question]

_**Agent (model composer-1, mode Agent)**_
[AI response]

[tool-use details]
[code changes]
[file reads]
```

## Advanced Usage

### Search Within Sessions
```bash
# Search for specific code patterns
grep -r "bg-slate-900" history/

# Find all sessions that modified a specific function
grep -r "updateFieldValue" history/
```

### Combine with Git
```bash
# Find sessions from a specific date range
ls history/ | grep "2026-01"

# Compare what was discussed vs what was committed
git log --since="2026-01-06" --until="2026-01-07"
```

### Export for Documentation
```bash
# Create a summary of all sessions
for file in history/*.md; do
    echo "## $(basename $file)"
    head -20 "$file"
    echo ""
done > session-summary.md
```

## Troubleshooting

### SpecStory Not Capturing?
- Check if `.specstory/history/` directory exists
- Verify Cursor AI is being used (SpecStory only captures Cursor sessions)
- Check file permissions on `.specstory/` directory

### Can't Find a Session?
- Use `./specstory-utils.sh search <term>` with different search terms
- Check the date range: `ls history/ | grep "2026-01"`
- Sessions are named by timestamp, so check dates

### Files Too Large?
- SpecStory captures full conversations including code
- Use `head` or `less` to view large files:
  ```bash
  head -100 history/large-file.md
  less history/large-file.md
  ```
