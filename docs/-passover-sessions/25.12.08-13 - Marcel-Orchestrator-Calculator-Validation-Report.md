### 2. Calculator Engine Deep Dive

Reviewed comprehensive calculator engine documentation to understand validation requirements:

**Engine Location:**
- File: `src/features/report-builder/store/reportBuilderStore.ts`
- Function: `runCalculations()` (lines 3181-3374, 193 lines)
- Status: Production-ready, validated against Valcre Excel

**11-Step Calculation Flow:**
1. Unit Mix Calculations (4 unit types)
2. Other Income (parking, laundry, other)
3. PGR = Rental Revenue + Other Income
4. Vacancy/Loss Calculations
5. EGR = PGR - Vacancy Loss
6. Operating Expenses (with 5 calc bases)
7. NOI = EGR - Total Expenses
8. Cap Rate Valuation (NOI ÷ Cap Rate)
9. Rounding to $10,000 (appraisal standard)
10. Post-Value Adjustments (CapEx, Leasing, Other)
11. Indicated Value → Syncs to RECON section

**5 Expense Calculation Bases:**
1. `per_unit` - Value × Number of Units (most expenses)
2. `per_sf` - Value × Total Square Footage (area-based)
3. `percent_egr` - EGR × Percentage (management fees - 4.25%)
4. `percent_pgr` - PGR × Percentage (rare)
5. `fixed` - Fixed dollar amount (admin, contracts)

**Critical Discovery:**
Management fees MUST use EGR (Effective Gross Revenue), NOT PGR (Potential Gross Revenue):
- **Correct:** 4.25% × $196,070 EGR = $8,333
- **Wrong:** 4.25% × $204,240 PGR = $8,680
- **Impact:** $347/year difference → ~$5,552 value impact at 6.25% cap

**Validation Test Data (North Battleford):**
- Property: 14 units, 11,480 SF
- Cap Rate: 6.25%
- PGR: $204,240
- Vacancy: 4% ($8,170 loss)
- EGR: $196,070
- Total OpEx: $84,621
- NOI: $111,449
- **Indicated Value: $1,780,000** ✅ Matches Valcre exactly
