## Technical Details

### Expense Calculation Flow
```
Store expects: per-unit annual values
Store calculates: value * totalUnits = total annual expense

Example:
- Input: calc-exp-taxes = 1168 (per unit)
- Calculation: 1168 * 16 units = $18,688 total taxes
```

### Navigation Flow (Fixed)
```
Test Input Dashboard (/test-input)
  -> Load Test Data (populates Zustand store)
  -> Click "Preview in Builder"
  -> navigate('/mock-builder') (preserves store state)
  -> MockReportBuilder checks sections.length
  -> If > 0: skip initialization, use existing data
  -> If = 0: initialize with mock data
```
