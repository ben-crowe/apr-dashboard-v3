## Critical Notes for HTML Template Verification

### All Tables Are Flattened Images
Every table in this Valcre report is rendered as a **flattened image**, not as actual Word table structures. This is critical for understanding how to implement dynamic data binding.

### Page Numbering System
- Physical pages 1-4: No footer numbers (cover, letter, TOC)
- Physical pages 5-74: Footer numbers 1-70
- Footer format: `{footer#}    {address}, {city}, {province} | File {file_number}`

### Section Headers Use Consistent Styling
- H1: Large blue text, underlined
- H2: Medium blue text
- Subsections: Bold blue text

### Photo Grid Layout
- Standard: 2 columns × 3 rows = 6 photos per page
- Each photo has caption below

### Comp Sheet Standard Format
Each of the 5 sales comparison comp sheets (Pages 54-58) follows identical structure:
- Property name and address header
- Sale Information section
- Income Analysis section
- Property photo (right side)
- Location map (right side)
- Unit Mix table
- Remarks section
