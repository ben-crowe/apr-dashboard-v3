## SET UP WORKBOOK

| Timeframe | Date logic, retrospective/current labels |
| Use Stabilization Tool | Enables stabilization analysis pages |
| Select Workbook | Which calculation workbook links |
| Select Modules | Which approach sections appear |

## Notable Features

| Feature | What It Does |
|---------|--------------|
| GET SIGNATURE button | Pulls appraiser signature image |
| GET GEOCODE button | Auto-fills lat/long from address |
| GET DETAILED GEOCODE | Enhanced location data |
| Color coding | Yellow=edit, Green=calc, Blue=dropdown |
| Cascading dropdowns | Property Type → Subtype → Modules |
| Default text | Auto-fills "None" statements if fields blank |
| Grammar helpers | Auto-correct singular/plural, articles |

- Photos / exhibits
