# Switch to Correct Google Cloud Project

**Issue:** You're in "My First Project" but need project `gen-lang-client-0262970848`

---

## Quick Fix

### Step 1: Switch Project

1. **Click the project dropdown** at the top (currently shows "My First Project")
2. **Search for:** `gen-lang-client-0262970848`
3. **Click on it** to select

**OR** if you don't see it:
- Click "All" to see all projects
- Look for a project with ID ending in `0262970848`
- Or create a new project with that ID

---

## Step 2: After Switching Projects

Once you're in the correct project (`gen-lang-client-0262970848`):

1. **Go to OAuth Overview** (you should already be there)
2. **Click "Create OAuth client"** button
3. **Or** go to "Clients" in the left menu to see existing clients

---

## Alternative: Use Existing Project

If you want to use "My First Project" instead:

1. **Click "Create OAuth client"** button
2. **Application type:** Desktop app
3. **Name:** APR Dashboard - bc@crowestudio.com
4. **Click "Create"**
5. **Download the JSON** and save as: `credentials/bc-crowestudio/credentials.json`

---

**Most Important:** Make sure you're in the project that matches your existing OAuth credentials!
