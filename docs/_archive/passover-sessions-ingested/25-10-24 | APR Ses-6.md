- Wrong field name in Valcre API
- Type mismatch (string vs number)

**Fix**:
1. Verify database column exists
2. Add retainer amount to Valcre sync payload with correct field name
3. Ensure proper data type (likely number, not string)

**Expected format**:
- `VAL251021 - Riverside Complex, 152 17th Avenue SW, Calgary`
- Drop: Province and Postal Code

**Files to check**:

```typescript
// BEFORE
```
