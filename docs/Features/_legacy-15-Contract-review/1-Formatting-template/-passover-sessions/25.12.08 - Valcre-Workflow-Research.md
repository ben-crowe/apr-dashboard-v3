## Key Finding: Image-Based Architecture Confirmed

### How Valcre Actually Works

**Three-Component System:**
```
Valcre Online (Database)
    ↓
Excel Workbook (Calculations)
    ↓ [AUTOMATED IMAGE CONVERSION]
Word Report (Embedded Images)
    ↓
PDF Export
```

**The "Sync" Process:**
- Appraiser clicks "Link & Sync" in Word
- Valcre converts Excel tables to images automatically
- Two quality settings:
  - **"Fastest Speed"** - Low-quality images (draft mode)
  - **"Highest Quality"** - High-quality images (slow, final version)
- Images embedded in Word document
- Result: Tables look like screenshots in final PDF

### Why This Explains Reference PDF

The North Battleford reference PDF shows **Valcre Excel screenshots** as flattened images because:
1. Valcre renders Excel tables as images during sync
2. Images are embedded (not live data)
3. Text extraction tools can't read image-table content
4. This is why gap analysis missed 89 image-tables initially

## Manual Pain Points That Still Exist

Despite Valcre's automation, appraisers must:

1. **Remove cell shading** from Excel tables before linking (manual button click)
2. **Manually link workbook to report** (file must be in same directory - USPAP requirement)
3. **Choose quality setting** each sync (draft fast vs final slow)
4. **Re-sync after every change** to workbook (not real-time)
5. **Edit red-highlighted sections** manually in Word
6. **Delete unused sections** manually
7. **Insert additional files/images** manually

### The Image Quality Tradeoff

**Problem:** Valcre forces choice between speed and quality
- Draft mode: Fast sync, blurry/pixelated images
- Final mode: Slow sync, better images (but still images, not live data)

**Why this matters:** Appraisers must render twice (draft review + final export)

## Valcre Workflow Details (Documented)

### Setup Phase
1. Create job in Valcre Online
2. Import property info from engagement letter
3. Search database for comparables
4. Add custom fields for specialized data

### Calculation Phase
1. Export job data to Excel Workbook
2. **MANUAL:** Click "Turn Workbook Shading Off" button
3. Perform calculations and analysis
4. Review calculated fields
5. Save workbook in same directory as report (USPAP requirement)

### Report Generation Phase
1. Open Word Report Template
2. Open Valcre Ribbon → Linked Files → Change File
3. **MANUAL:** Link workbook to report template
4. Select sync quality: "Fastest Speed" or "Highest Quality"
5. Click Update to generate report
6. **MANUAL:** Review red-highlighted sections requiring input
7. **MANUAL:** Delete unused sections marked in red
8. **MANUAL:** Insert additional files/images
9. **MANUAL:** Insert datasheets (PDFs) into Addenda

### Final Export Phase
1. Re-sync with "Highest Quality" for final version
2. Update Table of Contents
3. Export to PDF for delivery
