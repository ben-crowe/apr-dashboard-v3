## 🔍 DISCOVERIES

### What We Learned

1. **Typography Hierarchy Matters:** A 2pt difference between headers (11pt) and body (9pt) provides clear visual separation without feeling cramped
2. **Spacing Precision:** Reducing margins from 8-16px to 3-12px saves ~1 inch of vertical space without sacrificing readability
3. **Color Consistency:** Using the same blue (#003B7E) for both titles and table headers creates visual cohesion across the page
4. **Field Naming Pattern:** kebab-case conversion works well for multi-word fields (sk-avgrent-1bed, hbu-conclusion-vacant)
5. **Page Fit Strategy:** Tight spacing + smaller fonts + darker colors = more content per page without feeling crowded

### Key Insights

- **User preference established:** Headers bigger than body (but not excessively)
- **Rent Roll table format:** Compact, clean presentation with dark header bar
- **Market indicator fields:** CMHC data source with 8 key metrics for market context
- **HBU analysis structure:** 4-step framework (Legal → Physical → Financial → Productivity)
