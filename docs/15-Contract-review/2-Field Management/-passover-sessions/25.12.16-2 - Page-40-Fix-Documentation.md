## The Problems (What Was Broken)

### Problem 1: Wrong Field-Mapped Span Format
**What was wrong:**
```html
<!-- WRONG - Sample data as textContent -->
<span class="field-mapped" title="{{Market_Rent_1Bed_Comp1_UnitSize}}">650</span>
```

**What it should be:**
```html
<!-- CORRECT - Field ID as textContent, sample data in attribute -->
<span class="field-mapped" data-sample="650" title="{{Market_Rent_1Bed_Comp1_UnitSize}}">{{Market_Rent_1Bed_Comp1_UnitSize}}</span>
```

**Why this matters:**
- Toggle needs to show field IDs by default
- Sample data stored in `data-sample` attribute for toggle functionality
- JavaScript reads `data-sample` to display preview mode

---

### Problem 2: Missing Field-Mapped Spans Entirely
**What was wrong:**
```html
<!-- Page 40 tables had ZERO field-mapped spans -->
<td class="text-right">650</td>
<td class="text-right">$1,379</td>
```

**What it should be:**
```html
<!-- ALL numeric values wrapped in field-mapped spans -->
<td class="text-right"><span class="field-mapped" data-sample="650" title="{{Market_Rent_1Bed_Comp1_UnitSize}}">{{Market_Rent_1Bed_Comp1_UnitSize}}</span></td>
<td class="text-right"><span class="field-mapped" data-sample="$1,379" title="{{Market_Rent_1Bed_Comp1_RentUnit}}">{{Market_Rent_1Bed_Comp1_RentUnit}}</span></td>
```

**Why this matters:**
- Without field-mapped spans, no data binding possible
- Toggle won't work at all
- Dynamic data population impossible

---

### Problem 3: Missing <tr> Row Tags
**What was wrong:**
```html
<tbody>

        <td>1 Flat 1 Bed / 1 Bath</td>
        <td class="text-right">650</td>
        ...

</tbody>
```

**What it should be:**
```html
<tbody>
    <tr>
        <td>1 Flat 1 Bed / 1 Bath</td>
        <td class="text-right">650</td>
        ...
    </tr>
</tbody>
```

**Why this matters:**
- Browser can't render table properly without row tags
- Cells appear in wrong columns
- CSS styling breaks
- Screen readers can't parse table structure

---

### Problem 4: Broken Toggle JavaScript
**What was wrong:**
```javascript
// Using obsolete sampleData object
const sampleValue = sampleData[fieldName];
field.textContent = sampleValue;
```

**What it should be:**
```javascript
// Read from data-sample attribute
const sampleValue = field.getAttribute('data-sample');
field.textContent = sampleValue;
```

**Why this matters:**
- Old code relied on hardcoded JavaScript object
- Object was removed but toggle wasn't updated
- Toggle completely non-functional

---

### Problem 5: Inline Styles Making Tables Too Wide
**What was wrong:**
```html
<td style="border: 1px solid #ddd; text-align: right;">650</td>
```

**What it should be:**
```html
<td class="text-right">650</td>
```

**Why this matters:**
- Inline styles override CSS classes
- Tables become too wide to fit page
- Inconsistent with Page 44 reference standard
- Harder to maintain

### Fix 4: Update Toggle JavaScript (Commit df5589d)
```javascript
// OLD (broken)
previewToggle.addEventListener('change', function() {
    const isPreviewMode = this.checked;
    if (isPreviewMode) {
        const sampleValue = sampleData[fieldName];
        field.textContent = sampleValue;
    }
});

// NEW (working)
previewToggle.addEventListener('change', function() {
    const isPreviewMode = this.checked;
    if (isPreviewMode) {
        const sampleValue = field.getAttribute('data-sample');
        if (sampleValue) {
            if (!field.hasAttribute('data-original')) {
                field.setAttribute('data-original', field.textContent);
            }
            field.textContent = sampleValue;
        }
    } else {
        const originalText = field.getAttribute('data-original');
        if (originalText) {
            field.textContent = originalText;
        }
    }
});
```

## Common Mistakes to Avoid

❌ **WRONG:** First column (row labels) wrapped in field-mapped spans
```html
<td><span class="field-mapped"...>1 Flat 1 Bed / 1 Bath</span></td>
```

✅ **CORRECT:** First column as plain text
```html
<td>1 Flat 1 Bed / 1 Bath</td>
```

---

❌ **WRONG:** Field ID in data-sample, value as textContent
```html
<span class="field-mapped" data-sample="{{Market_Rent...}}">650</span>
```

✅ **CORRECT:** Value in data-sample, field ID as textContent
```html
<span class="field-mapped" data-sample="650" title="{{Market_Rent...}}">{{Market_Rent...}}</span>
```

---

❌ **WRONG:** Missing <tr> tags
```html
<tbody>
    <td>Label</td>
    <td>Value</td>
</tbody>
```

✅ **CORRECT:** Proper row wrappers
```html
<tbody>
    <tr>
        <td>Label</td>
        <td>Value</td>
    </tr>
</tbody>
```
