05 Role Dropdown — Independent / Trigger / Result If Trigger or Result chosen, Configure Logic button appears after save ▾
Role dropdown has 3 options: Independent (default — no logic), Trigger (drives a cascade), Result (set by a cascade).
If client picks Trigger or Result, the field saves with a cascade flag. A Configure Logic button appears on the saved row.

08 Edit Mode — Click existing row Only client-facing fields become editable. System fields stay read-only. ▾
Clicking an existing Valta Dashboard row enters edit mode. Editable: Label, Type (Control Type), Role, Dropdown List, Category (Valcre Location).
Read-only: #, Status, System, Field Name, Source, Valcre Field ID. These are system-managed.
Field Name does NOT change even if Label changes — it was set on creation. Dev team renames if needed.
