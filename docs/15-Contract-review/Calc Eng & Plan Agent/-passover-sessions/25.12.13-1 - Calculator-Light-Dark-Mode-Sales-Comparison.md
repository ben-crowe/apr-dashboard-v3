### 2. Sales Comparison Approach

**Files Created:**
- `src/features/calculator-demo/components/SalesComparisonPanel.tsx` (571 lines)

- `src/features/calculator-demo/components/SalesComparisonSection.tsx` (108 lines)

**Architecture:**
```
SalesComparisonSection (wrapper)
├── SalesComparisonPanel (5 comps + adjustments)
│   └── onIndicatedValueChange callback
└── ReconciliationPanel (weighted combination)
    └── receives salesIndicatedValue prop
```

**Key Formulas:**
```typescript
// Sales Comparison Panel
const calcPerUnit = (comp) => comp.salePrice / comp.units;
const calcNetAdjustment = (adj) =>
  adj.propertyRights + adj.financing + adj.conditionsOfSale +
  adj.marketConditions + adj.location + adj.size + adj.ageCondition + adj.other;
const calcAdjustedPerUnit = (comp, adj) => basePerUnit * (1 + netAdj / 100);
const indicatedValueSales = indicatedPerUnit * subjectUnits;

// Reconciliation Panel
const reconciledValue = (incomeValue * incomeWeight / 100) +
                        (salesIndicatedValue * salesWeight / 100);
```

**Features:**
- Subject property info section
- 5 comparable sales with editable fields
- 8-category adjustment grid (percentage-based)
- Auto-calculated $/Unit and $/SF
- Indicated value derivation
- Reconciliation with adjustable weights
- Final reconciled value with per-unit/per-SF metrics
