# Navigate to OAuth Consent Screen Configuration

**You're on:** OAuth Overview (just shows metrics)

**You need:** OAuth Consent Screen configuration

---

## Step 1: Click "Branding" in Left Menu

**In the left navigation menu, click:**
- **"Branding"** (this is where OAuth consent screen settings are)

---

## Step 2: Or Try "Audience"

**If "Branding" doesn't show consent screen settings, try:**
- **"Audience"** (this might have user/test user settings)

---

## Step 3: What You're Looking For

**Once you click "Branding" or "Audience", you should see:**

1. **Publishing status** (should say "Testing")
2. **Test users** section (should have `bc@crowestudio.com`)
3. **Scopes** or **Data Access** section
