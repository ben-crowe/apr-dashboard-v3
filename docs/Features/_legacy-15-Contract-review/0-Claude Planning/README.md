# Claude Planning - Working Folder

**Purpose:** Working folder for agent session summaries, scratch files, and in-progress work.

**This folder can be messy.** Important files should be moved to `0-APR-Domain-Core-Mgt/` when finalized.

---

## What Goes Here

- Session summaries and transcripts
- Scratch files and working notes
- In-progress analysis
- Temporary JSON exports
- Handoff files between sessions
- Any agent-generated files during work

---

## What Should Move to `0-APR-Domain-Core-Mgt/`

When a file becomes finalized and valuable for ongoing reference:
- Domain knowledge updates
- Finalized field mappings
- Curated workbook analysis
- Anything that should be kept clean and maintained

---

## Current Contents

This folder accumulates working files. Periodically review and:
1. Move valuable files to `../0-APR-Domain-Core-Mgt/`
2. Archive old session files to `../-passover-sessions/`
3. Delete obsolete scratch files

## For Agents

**Save session summaries here** with naming convention:
- `SESSION-{topic}-{date}.md`
- `SESSION-TRANSCRIPT-{topic}-{seq}.json`

**Before ending a session:**
1. Save summary to this folder
2. If file is valuable long-term, move to `../0-APR-Domain-Core-Mgt/`
3. Update domain knowledge file if needed
