### N8N Automation Workflows (5 Workflows Fully Designed)

#### Workflow 1: Missing Documents AI Review
**Trigger:** Form submission
**Process:**
1. AI reviews uploaded documents
2. Checks for missing critical documents
3. IF missing → Send follow-up email
4. Updates ClickUp with checklist

**Status:** 📋 Designed, awaiting N8N setup

#### Workflow 2: Google Drive Folder Creation
**Trigger:** Valcre job created (VAL# assigned)
**Process:**
1. Copy folder template
2. Rename: "VAL251019 - Property Name"
3. Create subfolders (01-Client-Documents, 02-Research, etc.)
4. Upload client files from Supabase
5. Update Supabase with folder URL
6. Add ClickUp comment

**Missing Info:**
- Folder template ID
- Parent folder ID
- OAuth credentials

#### Workflow 3: QuickBooks Invoice Generation
**Trigger:** LOE signed
**Process:**
1. Create/update customer in QuickBooks
2. Generate invoice
3. Email invoice to client
4. Update job status to "invoiced"
5. Update ClickUp task

**Missing Info:**
- QuickBooks credentials
- Company ID
- Tax settings
- Yumi's invoice template

#### Workflow 4: Payment Tracking & Job Activation
**Trigger:** QuickBooks payment received
**Process:**
1. Find job by invoice_id
2. Update status to "payment_received"
3. Assign to field appraiser in ClickUp
4. Activate job
5. Notify team

#### Workflow 5: Houski Data Validation (Most Detailed)
**Trigger:** Address entered in Section 3
**Process:**
1. Call Houski API (14 fields)
2. Call Google Maps API (place details)
3. Get Street View image
4. Claude AI cross-references:
   - Property type match?
   - Building size plausible?
   - Zoning logical?
   - Visual appearance match?
5. Calculate confidence score (0-100)
6. IF ≥80%: Populate fields
7. IF 50-79%: Populate with warning
8. IF <50%: Don't populate, flag for manual entry

**Validation Checks:**
```javascript
// Check 1: Property Type Match
if (houski.property_type === 'Residential' &&
    google.types.includes('shopping_mall')) {
  conflicts.push('Property type mismatch');
}

// Check 2: Building Size Plausibility
if (houski.building_size < 5000 &&
    google.types.includes('shopping_mall')) {
  conflicts.push('Building too small for mall');
}

// Check 3: Zoning Logic
if (houski.zoning.startsWith('R-') &&
    google.name.includes('Mall|Shopping')) {
  conflicts.push('Residential zoning for commercial');
}

// Check 4: Visual Analysis
const visualCheck = analyzeStreetView(image, houski);
if (visualCheck.conflicts.length > 0) {
  conflicts.push(...visualCheck.conflicts);
}
```
